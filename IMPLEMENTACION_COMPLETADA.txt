🚀 FUNCIONES PRINCIPALES
================================================================================

useCart()
├─ Acceder al carrito en cualquier componente
├─ Usar: const { cart, agregarAlCarrito } = useCart();

agregarAlCarrito(item)
├─ Agrega producto o incrementa cantidad
├─ Uso: agregarAlCarrito({ id, nombre, precio, cantidad })

abrirWhatsApp(items, total)
├─ Abre WhatsApp con mensaje automático
├─ Uso: abrirWhatsApp(cart.items, cart.total)

copiarMensajePedido(items, total)
├─ Copia mensaje al portapapeles
├─ Uso: await copiarMensajePedido(cart.items, cart.total)

convertirASOles(precioEuro)
├─ Convierte EUR → PEN
├─ Uso: convertirASOles("10,50€") // 37.80
